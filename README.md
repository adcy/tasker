# tasker
speeds-up task creation

# description

```
This application is a tool to rapidly create TFS tasks and synchronize them with wiki.

Usage:
  tasker [command]

Available Commands:
  completion  generate the autocompletion script for the specified shell
  create      Create new task
  help        Help about any command
  sync        Syncs wiki with tfs

Flags:
  -h, --help   help for tasker

Use "tasker [command] --help" for more information about a command.
```
